Datasets are stored in this folder as a binaray file in .dat format

1. "Numpy_train.dat" is the list of all traininig images converted to Numpy Array 
2. "Numpy_test.dat" is the list of all validation images converted to Numpy Array 
3. "Classes.dat" is the list of classes of both training and validation images
4. "Classes" is a list, 0th index contains list of classes of training images
5. 1th index contains list of classes of validation images.
