1. This folder contains the trained machine learning and deep learning models.
2. Simple Machine learning models are saved in ".sav" format like "svm.sav", "knn,sav", etc.
3. Deep learning models like CNN and AlexNet are stored in 2 files.
4. The model is stored in ".json" file and weights are stored in ".h5" file.
5. For example, AlexNet is saved as "AlexNet.json" and "AlexNet.h5".
